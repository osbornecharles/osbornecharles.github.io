Welcome!
This is a small website made to showcase my web development skills. The source code can be viewed here.
Technical Specs:
  • 100% pure Vanilla Javascript, handmade and locally sourced (by myself, with some help from the internet)
  • Served up by a static fileserver hosted by Github Pages
  • This website is completely static, has minimal overhead and almost a complete lack of third party libraries
  • The only third party library used is a page view counter. To avoid having to use a code bundler and its associated overhead, this library is served from a CDN
  • I'm an avid CSS flexbox fan - expect a lot of display: flex
  • Cool code alert: background pictures can be swapped out with others of different aspect ratios, it won't break the layout
